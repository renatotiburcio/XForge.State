# XForge.State

Reactive state management: IState<T>, Subscribe, DerivedState, StateStore with Blazor support.

## Packages

- `XForge.State`
- `XForge.State.Blazor`


## Dependencies

- [XForge.Infrastructure](https://github.com/renatotiburcio/XForge.Infrastructure)

## Getting Started

```powershell
dotnet add package XForge.State
```

## Build

```powershell
dotnet build --configuration Release
```

## Test

```powershell
dotnet test --configuration Release
```
